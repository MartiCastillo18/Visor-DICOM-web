# Prototipo Visor DICOM

Este proyecto consiste en un visor de imágenes médicas (DICOM) integrado con un modelo de IA
para zonas toraxicas. 

Nota: Este visor corresponde a la primera versión del proyecto.  
No se utilizó en su totalidad en el producto final, ya que la IA no era compatible con el visor original y fue necesario reescribir el código.  
Aun así, se conserva como prototipo funcional y parte del proceso de desarrollo.

## Tecnologías
- XAMPP
- PHP
- MySQL
- JavaScript
- HTML
- CSS

## Instrucciones
1- Clonar el repositorio
2- Instalar XAMPP
3- Crear la base de datos en MySQL
4- Copiar el proyecto en la carpeta htdocs de XAMPP
5- Iniciar Apache y MySQL desde el panel de XAMPP
6- Abrir el navegador y escribir:
http://localhost/visor_web/
